# Multitool PWA

Multitool PWA - это идеальное решение для тех, кто ценит простоту и функциональность. Оно заменяет множество отдельных приложений, предоставляя под рукой базовый набор полезных инструментов. В будущем планируется расширять коллекцию функций как для повседневных задач, так и для профессиональных разработчиков.

## Основные функции на данный момент:
- Погода - прогнозы для любого города
- Файловый менеджер - загрузка и хранение файлов
- Debug панель - мониторинг PWA и тестирование

## PWA возможности:
- Установка как нативное приложение
- Полный оффлайн режим
- Push-уведомления
- Адаптивный дизайн
- Быстрая загрузка

## Ключевые технологии:
- **Frontend:** React 19.2.0 + JavaScript ES6+
- **PWA:** Service Worker, Web App Manifest, Workbox
- **APIs:** Open-Meteo, WeatherAPI, File API, Notification API
- **CI/CD:** GitHub Actions, ESLint, Prettier, Jest
- **Деплой:** Vercel
